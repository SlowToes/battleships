## Battleships Online

A multiplayer Battleships game built with a **Spring Boot backend** and a **TypeScript frontend**.

### Tech Stack
- **Backend:** Spring Boot (Java)
- **Frontend:** TypeScript
- **Database:** PostgreSQL
- **Real-time Communication:** WebSockets

### Deployment
- **Backend & Database:** Render
  https://battleships-v2y0.onrender.com

- **Frontend:** Vercel
- https://battleships-lime-beta.vercel.app/
